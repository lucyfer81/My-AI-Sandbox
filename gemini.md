项目说明
项目概述
本目录用于部署个人博客，使用 Hugo 框架构建。
部署流程
代码管理：完成修改后，通过 Git 推送到 GitHub。
自动部署：Cloudflare 会自动抓取更新并部署。
Python 环境
虚拟环境：调用 Python 时，请使用虚拟环境路径：.venv/bin/。
系统环境
操作系统：Ubuntu 24.04
硬件配置：双核 4G 内存
特性：无 GUI，仅支持命令行操作。
反向代理
访问方式：通过反向代理访问博客，测试地址为 http://blog_test.vftl.site。
注意：运行 Hugo 服务器时，请注意反向代理的限制。
语言要求
沟通语言：全程使用中文进行沟通。
